# Notepad (macOS)

Editor de texto plano ultraminimalista para macOS con guardado automatico.

## Requisitos
- macOS 13 o superior
- Xcode Command Line Tools (`xcode-select --install`)

## Compilar
```
./build.sh
```

Se genera `build/Notepad.app`.

## Usar
- Abrir la app con doble click.
- Copiar, pegar y cortar funcionan con los atajos estandar (Cmd+C, Cmd+V, Cmd+X).
- El contenido y el tamano de letra se guardan automaticamente.
- Ajustar el tamano con los botones en la barra superior.
